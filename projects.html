Projects

Here are some of the projects I've been working on. Most are open source and available on GitHub.

Flask-RESTful

Python ⭐ 6.8k

Simple framework for creating REST APIs with Flask. Provides a coherent collection of decorators and tools to describe your API and expose its documentation properly.

GitHub Documentation PyPI

Click

Python ⭐ 14.2k

A Python package for creating beautiful command line interfaces in a composable way with as little code as necessary. It's highly configurable but comes with sensible defaults out of the box.

GitHub Documentation PyPI

Werkzeug

Python ⭐ 6.3k

A comprehensive WSGI web application library. It began as a simple collection of various utilities for WSGI applications and has become one of the most advanced WSGI utility libraries.

GitHub Documentation PyPI

Jinja2

Python ⭐ 9.8k

A modern and designer-friendly templating language for Python, modelled after Django's templates. It is fast, widely used, and secure with the ability to sandbox execution.

GitHub Documentation PyPI

ItsDangerous

Python ⭐ 2.7k

A Python library for passing data through untrusted environments (like cookies or URLs) while ensuring the data hasn't been tampered with. It uses HMAC to sign the data.

GitHub Documentation PyPI

Babel

Python ⭐ 1.1k

A collection of tools for internationalizing Python applications. It includes a message extraction system, a translation system, and a locale-dependent formatting system.

GitHub Documentation PyPI
